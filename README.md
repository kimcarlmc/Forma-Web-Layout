# FORMA – Responsive One-Page Website

A responsive one-page website built with **HTML**, **CSS**, and **JavaScript**, inspired by a **Figma AI-generated mockup**. This project was created as a front-end development exercise to strengthen my ability to translate a design into clean, responsive code while improving my understanding of modern CSS layout techniques.

**Live Demo:** https://forma-web-layout.vercel.app

---

## About the Project

FORMA is a static landing page developed as part of my front-end learning journey. The primary objective of this project was to practice building responsive layouts from a design mockup while reinforcing core web development fundamentals.

The project emphasizes semantic HTML structure, responsive design principles, and the use of CSS Grid to create a clean, adaptable layout across different screen sizes.

---

## Features

* Fully responsive layout
* Built from a Figma AI-generated mockup
* CSS Grid-based page layout
* Semantic HTML structure
* Responsive navigation menu
* Lightweight static website
* Deployed on Vercel

> **Note:** This website is currently a static landing page. The only interactive functionality implemented is the responsive navigation menu.
